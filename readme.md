## Описание проекта автоматизации
#### 1. Цель:
>Демонстрация навыков и автоматизация ключевых модулей сайта Кинориум для обеспечения стабильности работы, минимизации ошибок и улучшения пользовательского опыта.
#### 2. Задачи:
>- Проанализировать структуру и основные пользовательские      сценарии сайта Кинориум.
>- Провести функциональное и нефункциональное тестирование приложения
>- Провести анализ результатов автоматизации и оценить вклад автотестов в повышение качества продукта.
>- Сформулировать выводы и рекомендации по дальнейшему развитию автоматизации для проекта.

---

#### 3. Объектиы автоматизации:
- Регистрация и авторизация
- Управление профилем
- Поиск и рекомендации фильмов/сериалов
- Социальные функции (комментарии, рейтинги, списки просмотра)
- Адаптивный дизайн (мобильная и десктопная версии)

---

#### 4. Инструменты:
- Тестирование UI: 
  - Selenium 
  - Playwright
- API-тесты: 
  - Postman
  - insomnia
- Нагрузочное тестирование: 
  - JMeter
  - Locust

---

#### 5. Ожидаемые результаты:
- Покрытие процента критических сценариев автотестами
- Сокращение времени ручного тестирования

## Чек лист:
- Навигация по сайту
  - Проверить переход по основным разделам сайта («Премьеры», «Все фильмы», «Сериалы», «Лента», «Видео»).
  - Проверить возврат на главную страницу кликом по логотипу «Кинориум».
  - Проверить переход в раздел «Donate».
Поиск
  - Проверить поиск фильма с корректным названием.
  - Проверить поиск фильма с опечаткой в названии.
  - Проверить поиск по имени актёра.
  - Проверить работу фильтров поиска (фильмы / сериалы / мультфильмы и всплывающие меню).
  - Проверить работу чекбоксов фильтрации в разделе видео.
  - Проверить работу всплывающих меню фильтров в разделе «Премьеры» (жанры / страны / годы).
- Регистрация и авторизация
  - Проверить успешную регистрацию новой учётной записи.
  - Проверить успешную авторизацию с валидными данными.
  - Проверить переход в профиль при клике на аватар.
- Действия на странице фильма
  - Проверить работу кнопок активности («Не буду смотреть», «Буду смотреть», «Смотрю», «Оценка»).
- Профиль пользователя
  - Проверить переход в раздел «Бейджи».
  - Проверить копирование ссылки на профиль (кнопка копирования).
  - Проверить переход в раздел «Настройки».
- Настройки
  - Проверить работу чекбоксов в разделе «Основное».
  - Проверить изменение никнейма и сохранение результата.
  - Проверить резервное копирование профиля и получение уведомления о отправке файла.
- Завершение сессии
  - Проверить корректный выход из учётной записи и возврат на главную страницу.
